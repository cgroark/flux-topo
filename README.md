# Flux Topo with Astro Decap CMS

A fully customizable React-based static site with a git-based CMS content editor. Built with Astro and Decap CMS.

## Overview

This is website for a small business for potential clients. It also shows the possibilities of static site generation with a git-based CMS admin panel allowing content and image editing. Built with Astro and Decap CMS (Netlify CMS) and hosted with Netlify, this allows a free to deploy, free to host app that can be edited without code in a CMS editor.

### Features
 - Single page app with sections and scrolling navigation
 - Admin git-based CMS panel to edit all content and images

### Technology
 - Astro
 - Decap CMS
 - React, HTML, CSS
 - Fully editable git-based CMS
 - Netlify Identity auth and hosting
